# Localisation pharmaceutique 🏥

Bienvenue sur le projet **Localisation pharmaceutique**. Cette application web a pour objectif de faciliter la recherche et la géolocalisation des pharmacies à proximité pour les utilisateurs.

## 🚀 Fonctionnalités
- **Recherche intuitive :** Trouvez rapidement les pharmacies autour de vous.
- **Interface responsive :** Utilisation fluide sur mobile et desktop.
- **Gestion des urgences :** Accès facilité aux services de garde.
- **Dashboard utilisateur :** Espace personnel pour gérer vos besoins.

## 🛠️ Technologies utilisées
- **Frontend :** HTML5, CSS3, JavaScript.
- **PWA (Progressive Web App) :** Utilisation de `manifest.json` et `sw.js` pour une expérience hors-ligne.

## 📁 Structure du projet
- `localisation.html` : Page principale de recherche.
- `dashboard.html` : Espace utilisateur.
- `urgence.html` : Section dédiée aux urgences.
- `sw.js` : Service Worker pour la mise en cache.

## 💻 Installation
1. Clonez le dépôt : `git clone https://github.com/ablamkoumekpo28-maker/pharmac-localisation.git`
2. Ouvrez `localisation.html` dans votre navigateur préféré.
